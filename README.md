Aplicación Android para detección de melanomas mediante procesamiento de imágenes con OpenCV.
Challenge 4, Tópicos Avanzados de Inteligencia Artificial

Características
- Captura de imágenes con cámara o galería
- Procesamiento de imágenes con OpenCV
- Análisis de métricas (radio promedio, irregularidad)
- Clasificación: Benigno o Maligno
